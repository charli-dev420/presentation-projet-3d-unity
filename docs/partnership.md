# Partnership And Open Needs / Partenariats et besoins ouverts

[EN](#english) | [FR](#francais)

## English

### Open Collaboration

This project is open to discussions around Unity asset workflows, technical-art QA, mobile constraints, dataset review, 2.5D/facade preparation, ComfyUI handoff, product workflow design, documentation, funding, mission work, and roles.

The useful starting point is a concrete question: source quality, Splat Face candidate criteria, Unity import behavior, product workflow state, or local asset readiness.

### Strong Support Areas

Unity reviewers can help define import criteria, scale expectations, material checks, and mobile budgets.

Technical artists can help judge whether facade or 2.5D candidates are actually useful in a scene.

Dataset reviewers can help define what makes a source worth processing.

Product collaborators can help make jobs, artifacts, states, and client summaries easier to understand.

Funders or mission leads can support the next proof loop: reviewed source set, Splat Face route, Unity check, and written decision.

Public contact route: [GitHub - Unicorn Who Dev](https://github.com/charli-dev420).

## Francais

### Collaboration Ouverte

Le projet est ouvert aux discussions autour des workflows assets Unity, QA technical-art, contraintes mobile, revue dataset, preparation 2.5D/facade, handoff ComfyUI, design workflow produit, documentation, financement, missions et postes.

Le bon point de depart est une question concrete: qualite source, criteres candidat Splat Face, comportement import Unity, etat workflow produit ou readiness asset locale.

### Zones De Support Fortes

Les reviewers Unity peuvent aider a definir criteres import, attentes d'echelle, controles materiaux et budgets mobile.

Les technical artists peuvent aider a juger si les candidats facade ou 2.5D sont reellement utiles en scene.

Les reviewers dataset peuvent aider a definir ce qui rend une source exploitable.

Les collaborateurs produit peuvent aider a rendre jobs, artefacts, etats et resumes client plus comprehensibles.

Financeurs ou responsables mission peuvent soutenir la prochaine boucle preuve: lot source revu, route Splat Face, controle Unity et decision ecrite.
