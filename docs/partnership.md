# Partnership Brief / Fiche partenariat

## Francais

### Besoin

Le projet 3D Unity cherche a rendre les workflows assets IA plus fiables: selection d'images, evaluation, generation, normalisation, packaging et import Unity.

### Apports recherches

- Retours de production Unity.
- Expertise pipeline 3D mobile.
- Tests sur assets temps reel.
- Financement pour stabilisation et validation editor.
- Collaboration autour dataset review, ComfyUI, Trellis, Flux et packaging Unity.

### Ce qui est partage publiquement

Une presentation de la demarche, pas les assets, modeles, workflows prives ou projets Unity internes.

## English

### Need

The 3D Unity project aims to make AI asset workflows more reliable: image selection, evaluation, generation, normalization, packaging, and Unity import.

### Looking for

- Unity production feedback.
- Mobile 3D pipeline expertise.
- Realtime asset testing.
- Funding for stabilization and editor validation.
- Collaboration around dataset review, ComfyUI, Trellis, Flux, and Unity packaging.

### What is shared publicly

A presentation of the approach, not private assets, models, workflows, or internal Unity projects.
