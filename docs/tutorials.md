# Tutorials / Tutoriels

[FR](#francais) | [EN](#english)

## Francais

### Tutoriel 1 - Comprendre un dataset asset

Un dataset asset n'est pas seulement un dossier d'images. Il doit permettre de savoir quoi generer, avec quelles limites, et comment verifier le resultat. Les criteres de base sont: lisibilite, coherence, absence de bruit bloquant, orientation, usage cible et provenance.

### Tutoriel 2 - Evaluer un asset 2.5D

1. Identifier la source visuelle.
2. Verifier les maps ou rendus attendus.
3. Controler silhouette, profondeur, echelle et densite.
4. Evaluer l'integration dans Unity.
5. Documenter ce qui est valide et ce qui reste fragile.

### Tutoriel 3 - Preparer un import Unity

Avant import, confirmer: nommage, pivot, echelle, orientation, bounds, materiaux, poids, textures, collisions si necessaire, et statut de validation.

### Tutoriel 4 - Lire une preuve qualite

Une preuve utile indique: source, scenario, resultat attendu, resultat observe, limites, artefacts partageables et statut. Les logs bruts ne sont pas une preuve publique acceptable sans redaction.

## English

These tutorials explain the public process: understand asset datasets, evaluate 2.5D outputs, prepare Unity import, and read quality evidence without exposing private code or raw logs.
