# User Flows / Flux utilisateur

[FR](#francais) | [EN](#english)

## Francais

### 1. Operateur dataset

1. Rassembler les images candidates.
2. Evaluer clarte, cadrage, coherence, utilite et risques.
3. Marquer ce qui est exploitable, a reprendre ou a rejeter.
4. Exporter un lot pret pour generation ou bake.
5. Documenter provenance et limites.

### 2. Artiste ou integrateur Unity

1. Lire le type d'asset attendu: 2.5D, facade, prefab, scene ou element test.
2. Verifier pivot, echelle, bounds, orientation et densite.
3. Importer dans une scene Unity.
4. Tester lisibilite, performance et collision si applicable.
5. Remonter les corrections necessaires.

### 3. Acheteur ou collaborateur

1. Lire le one-pager.
2. Verifier quels repos sont publics ou prives.
3. Lire les preuves et limites.
4. Choisir un scenario de demo: dataset review, bake 2.5D, generation, import Unity.
5. Definir les criteres de succes avant d'ouvrir un acces prive.

## English

The user flow is: evaluate candidate images, prepare a dataset, generate or bake assets, normalize for Unity, import into a scene, then evaluate usability, performance, and provenance.
