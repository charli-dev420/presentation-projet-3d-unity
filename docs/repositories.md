# Repositories Covered / Repos couverts

[EN](#english) | [FR](#francais)

## English

This showcase represents several real 3D, Unity, dataset, and asset-pipeline repositories. It publishes presentation material, diagrams, quality criteria, and reading paths.

**Splat Face / Splat Facade Baker** is the main project line. Public route: [Unicorn Who Dev / splat-facade-baker](https://github.com/charli-dev420/splat-facade-baker).

**Dataset ReviewEval** is the source-review surface. Public route: [Unicorn Who Dev / datasetvieweval](https://github.com/charli-dev420/datasetvieweval).

**CodexUnity / CodexToUnity** is the Unity handoff bridge. Public route: [Unicorn Who Dev / codextounity](https://github.com/charli-dev420/codextounity).

**Mob'ia / ccomf-unity** is the product layer around jobs, profiles, artifacts, and clients.

**LocalAssetFactory** is the local preparation and validation loop for asset candidates.

### Sharing Rule

Public repository links can be used directly. Internal project areas are represented through product framing, diagrams, review criteria, and written summaries.
