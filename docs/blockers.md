# Readiness Notes / Notes de readiness

[EN](#english) | [FR](#francais)

## English

### Open Gates

The next gates are practical and positive. Splat Face needs one strong reviewed source route, one asset candidate review, and one Unity-facing checklist result. Dataset ReviewEval can supply the source decision. CodexUnity and LocalAssetFactory can supply the handoff language. Mob'ia can supply the product-readable state.

The goal is not to list limitations. The goal is to show exactly what kind of support would make the next proof stronger.

### Useful Next Evidence

- A source review note with keep/fix/reject decisions.
- A Splat Face candidate note with facade or 2.5D criteria.
- A Unity checklist result for import, scale, pivot, bounds, materials, and mobile weight.
- A product workflow note showing job, artifact, state, and next action.
