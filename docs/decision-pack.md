# Decision Pack / Dossier de decision

[EN](#english) | [FR](#francais)

## English

### Recommended Next Decision

The next strong decision is to pick one reviewed source set and one Splat Face route, then produce a Unity-facing review note. That is enough to show whether the project direction is useful without spreading the work across too many surfaces.

### For A Funder

Support should turn the pipeline into repeatable proof: source criteria, candidate preparation, Unity checks, documentation, and a small library of allowed validation cases.

### For A Unity Partner

The useful entry point is a technical-art review around scale, pivot, bounds, materials, texture readability, import behavior, and mobile cost.

### For A Product Collaborator

The useful entry point is the workflow surface: profile, job, artifact, state, client, review note, and next action.

### Decision Questions

Which source type should prove the route first? Which Unity constraint matters most? Which artifact state should a user see? Which support area creates the strongest next proof?

## Francais

### Prochaine Decision Recommandee

La prochaine decision forte est de choisir un lot source revu et une route Splat Face, puis produire une note de revue orientee Unity. C'est suffisant pour montrer si la direction projet est utile sans disperser le travail sur trop de surfaces.

### Pour Un Financeur

Le support doit transformer le pipeline en preuve repetable: criteres source, preparation candidat, controles Unity, documentation et petite bibliotheque de cas de validation autorises.

### Pour Un Partenaire Unity

Le point d'entree utile est une revue technical-art autour d'echelle, pivot, bounds, materiaux, lisibilite texture, comportement import et cout mobile.

### Pour Un Collaborateur Produit

Le point d'entree utile est la surface workflow: profil, job, artefact, etat, client, note de revue et prochaine action.

### Questions De Decision

Quel type de source doit prouver la route en premier ? Quelle contrainte Unity compte le plus ? Quel etat artefact un utilisateur doit-il voir ? Quelle zone de support cree la prochaine preuve la plus forte ?
