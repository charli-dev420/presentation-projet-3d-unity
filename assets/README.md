# Assets

This folder contains presentation assets for the Splat Face / 3D Unity showcase.

## Included

- `project-banners/` - banners for Splat Face, Mob'ia, and the Unity workflow.
- `diagrams/` - pipeline map, QA checklist, proof ladder, and status matrix.
- `brand/` - Unicorn Who Dev / DAW Core shared identity marks.
- `one-pager.svg`, `proof-dashboard.svg`, `qa-matrix.svg` - visual summaries used by the docs.

## Rule

Only add assets that help explain the project without exposing secrets, tokens, raw logs, local paths, unreleased builds, model files, rights-sensitive source material, unpublished datasets, generated meshes, or production infrastructure details.
